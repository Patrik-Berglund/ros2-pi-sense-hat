Specification
Pressure/temperature sensor: STMicro LPS25HB
• 260 to 1260 hPa absolute pressure range
• 24-bit pressure data output
• 0 to +65°C accurate temperature measurement range
(±2°C)
• 16-bit temperature data output

Humidity temperature sensor: STMicro HTS221
• 0 to 100% relative humidity range
• 15 to 40°C accurate temperature measurement range
(±0.5°C)
• 16-bit data output

IMU: STMicro LSM9DS1
Accelerometer/gyroscope/magnetometer
• ±16g acceleration measurement range
• ±16 gauss magnetometer measurement range
• ±2000dps gyroscope measurement range
• 16-bit resolution for each measurement channel

Colour sensor: AMS TCS34725 RGB colour and brightness sensor
• I2C interface (address 0x29)
• RGBC color detection (Red, Green, Blue, Clear)
• Ambient light sensing
• Programmable gain and integration time
• Note: Hardware uses TCS34725, not TCS3400 as originally specified

LED matrix: 8×8 RGB LED display
• Driver: STMicroelectronics LED2472G
• 64 RGB LEDs (192 individual LED channels)
• I2C or SPI control
• PWM brightness control

Joystick: 5-button miniature joystick with up, down, left, right,
and middle-click
